# GEO 计费系统 V1.1 站群模块 — 页面清单

> **用途**：把 `sitefarm/` 下的功能结构图整理成可直接用于画原型 / 开发的页面清单  
> **日期**：2026-07-09  
> **范围**：超级管理员端、SAAS 租户端、终端用户端的站群相关模块

---

## 一、页面总览

| 端口 | 模块 | 页面 | 建议路由 / 原型文件 | 优先级 |
|------|------|------|---------------------|--------|
| 超级管理员端 | SAAS 用户管理 | SAAS 用户列表 + 站群权限配置 | `/admin/saas-users` | P0 |
| 超级管理员端 | 网站模板管理 | 模板列表 / 新增 / 编辑 / 预览 | `/admin/sitefarm-templates` | P0 |
| 超级管理员端 | TDK 变量管理 | 变量列表 / 新增 / 编辑 | `/admin/tdk-variables` | P1 |
| SAAS 租户端 | 基础资料管理 | 客户基本信息列表 / 编辑 | `/tenant/sitefarm-profiles` | P0 |
| SAAS 租户端 | 网站模板管理 | 可用模板列表 / 预览 / 分配用户 | `/tenant/sitefarm-templates` | P0 |
| SAAS 租户端 | 训练词管理 | 训练词列表 / 图片库 / 知识库 / 标题模板 | `/tenant/training-words` | P0 |
| SAAS 租户端 | TDK 模板管理 | TDK 模板列表 / 新增 / 编辑 / 实时预览 | `/tenant/tdk-templates` | P0 |
| SAAS 租户端 | 快通管理 | 快通站点列表 / 创建 / 编辑店铺 | `/tenant/kuaitong` | P0 |
| SAAS 租户端 | GEO 文章管理 | 文章发布统计 / 文章列表 | `/tenant/geo-articles` | P1 |
| 终端用户端 | 基本信息 | 企业基本信息维护 | `/enduser/sitefarm-profile` | P0 |
| 终端用户端 | 快通管理 | 我的快通站点列表 / 编辑店铺 | `/enduser/kuaitong` | P0 |

## 二、超级管理员端

### 2.1 SAAS 用户管理 — 站群权限配置

**入口**：`/admin/saas-users`

#### 列表页字段

| 字段 | 组件 | 必填 | 说明 |
|------|------|------|------|
| 用户名称 | 文本 | - | SAAS 租户名称 |
| 已授权模板数 | 数字 | - | 该租户可使用的网站模板数量 |
| 状态 | 标签 | - | 正常 / 欠费 / 冻结 |
| 操作 | 按钮组 | - | 查看详情、编辑、冻结/解冻 |

#### 详情页 / 弹窗 — 站群模块权限

| 字段 | 组件 | 必填 | 说明 |
|------|------|------|------|
| 训练词管理 | 开关 | 否 | 是否开通 |
| 快通管理 | 开关 | 否 | 是否开通 |
| 模板管理 | 开关 | 否 | 是否开通 |
| TDK 模板 | 开关 | 否 | 是否开通 |
| 文章管理 | 开关 | 否 | 是否开通 |

#### 已授权模板

| 字段 | 组件 | 必填 | 说明 |
|------|------|------|------|
| 已授权模板 | 标签列表 | - | 展示已授权给该租户的网站模板 |
| 添加模板 | 弹窗 | - | 从模板库中选择并添加 |
| 删除模板 | 按钮 | - | 移除某个已授权模板 |

#### 操作
- 查看详情、编辑、保存、取消

#### 异常情况
- 空数据：暂无 SAAS 用户
- 查询无结果：未找到符合条件的用户

| 交互动效 | 下拉选择 | 否 | 中度动效 / 其他 |

- 新增模板、编辑模板、删除模板、预览、保存、取消

## 六、已确认规则

| 问题 | 确认结果 |
|------|----------|
| 终端用户端是否需要「创建快通」功能？ | 用户端只能查看快通站点，不能创建；只能编辑基础资料，不能编辑快通网站内容 |
| 已发布快通站点的访问形式 | 点击访问后在新标签页打开 |
| GEO 文章的内容文件 | 在线预览 |
| 超管端「文章管理」开关控制范围 | 只控制租户端，用户端不设置权限 |
| 城市选择数据粒度 | 只到城市，不到区县 |

> 基于以上确认，终端用户端「快通管理」页面移除「编辑店铺」操作，仅保留「复制店铺标题」和「访问快通站点」。
