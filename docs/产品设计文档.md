> 版本：V1.0 | 日期：2026-06-04 | 阶段：第一期

## 一、产品概述

### 1.1 产品定义

品牌监控SAAS系统是一款面向 **Generative Engine Optimization（生成式引擎优化）** 的一站式SaaS平台，覆盖品牌监控 → 内容生成 → 多渠道分发 → 效果量化的全链路闭环。

### 1.2 核心价值主张

| 角色 | 核心价值 |
|------|----------|
| **超级管理员** | 灵活售卖模块化SaaS能力，管理SAAS用户全生命周期，收取年费+增值费 |
| **SAAS租户** | 拥有独立的GEO运营平台，管理代理商渠道、品牌监控、AI内容生产和分发 |
| **代理商** | 独立定价签约，将SAAS租户的GEO能力转售给终端客户，赚取差价（一级代理可发展二级代理） |
| **终端用户** | 轻量级报表查看门户，透明掌握品牌GEO优化效果 |

### 1.3 差异化亮点

- 国内外AI搜索引擎全覆盖监控（ChatGPT、Gemini、豆包、Kimi、DeepSeek等）
- 模块化售卖 + 代理商二级分销体系，同时服务品牌企业和营销机构两个市场
- AI内容生成 → 媒体投稿 → 效果追踪的业务闭环
- 四个端口统一设计语言，OEM白标支持

### 1.4 客户画像

- **品牌方/企业市场部** — 监控和优化自身品牌在AI搜索引擎中的表现
- **SEO/营销服务机构** — 通过代理商渠道为客户提供GEO优化服务
- 平台同时面向以上两类客户，两者兼有

## 二、业务模式

### 2.1 计费模式

**第一期**：按模块拆分售卖，SAAS用户按需选购模块（品牌监控、AI文章生成、媒体投稿、采集任务、企业知识库等）

**后续**：叠加版本分层策略，固定功能模块打包售卖（如基础版/专业版/企业版）

### 2.2 代理商分销模型

采用**独立定价、独立签约**模式：

```
SAAS租户 → 设定产品套餐原始价格
    ↓
代理商 → 在原始价格基础上调整定价，与终端用户独立签约
    ↓
代理商 → 提交项目订单到SAAS租户端
    ↓
SAAS租户 → 执行GEO优化服务
    ↓
终端用户 → 查看效果报表
```

- 一级代理可发展二级代理
- 代理商自负盈亏，赚取差价

### 2.3 终端用户定位

终端用户为**代理商的客户联系人**，平台提供轻量级报表查看门户，主要操作包括：
- 查看品牌GEO优化效果报表
- 管理企业知识库素材（供SAAS端引用）
- 查看财务充值/消费记录

## 三、系统架构

### 3.1 架构模式

**统一平台 + 多租户数据隔离**（方案一）

- 统一技术底座，4个端通过角色权限隔离数据和功能
- 统一数据库，通过 `tenant_id` 行级数据隔离
- 四层层级关系：平台 → SAAS租户 → 代理商（一级/二级） → 终端用户

## 五、全局 UI 设计规范

### 5.1 设计风格

**简洁现代风** — 浅色为主，大留白，圆角卡片，轻量配色，科技感强。

### 5.2 统一设计语言

四个端口使用同一套UI组件和配色风格，仅通过菜单权限和功能区分，保持视觉一致性，降低开发和维护成本。

### 5.3 菜单导航

**左侧纵向菜单**（经典后台布局），顶部显示系统名称/Logo、全局搜索、消息通知、用户头像入口。

### 5.4 终端用户端定位

采用**后台管理风格**，与其他三端视觉一致，不做独立门户。

### 5.5 配色参考

| 元素 | 说明 |
|------|------|
| 主色调 | 科技蓝（#2563EB / #3B82F6） |
| 侧边栏 | 深色（#1E293B）或极浅灰（#F8FAFC） |
| 内容区 | 白色（#FFFFFF）+ 浅灰卡片（#F1F5F9） |
| 功能色 | 成功绿、警告橙、危险红、信息蓝 |

## 六、关键业务流程图

### 6.1 SAAS用户采购流程

```
超级管理员创建SAAS用户
    → 配置开通模块 & 有效期
    → SAAS用户登录激活
    → 可按需增购模块（产生账单）
    → 到期提醒 → 续费或冻结
```

### 6.2 代理商分销流程

```
SAAS租户设置产品套餐原始定价
    → 代理商查看套餐，调整定价
    → 代理商添加终端客户
    → 代理商与客户签署合同
    → 代理商提交项目订单到SAAS租户
    → SAAS租户审批 → 执行服务
    → 终端用户查看效果报表
```

### 6.3 终端用户服务流

```
代理商签约客户 → 提交订单
    → SAAS租户建立监控项目 → 配置关键词和平台
    → 采集AI搜索结果数据
    → 从客户知识库引用素材 → AI生成文章
    → 媒体投稿分发
    → 生成效果报表 → 终端用户查看
```
