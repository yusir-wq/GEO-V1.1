## 1. 框架与布局

### 1.1 整体布局结构

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          Header (深蓝渐变)                               │
│  [Logo + 品牌名]                              [用户头像 + 用户名 ▼]    │
├──────────────┬──────────────────────────────────────────────────────────┤
│              │                                                          │
│   侧边栏     │                    页面标题区域                         │
│  (固定宽度)  │    ┌──────────────────────────────────┐  [← 返回订单列表]│
│              │    │  大标题                          │                  │
│  - 菜单项1   │    │  副标题                          │                  │
│  - 菜单项2   │    └──────────────────────────────────┘                  │
│    - 子菜单  │                                                          │
│  - 菜单项3   │    ┌─────────────────────────────────────────────┐       │
│              │    │  卡片1                                       │       │
│              │    └─────────────────────────────────────────────┘       │
│              │    ┌─────────────────────────────────────────────┐       │
│              │    │  卡片2                                       │       │
│              │    └─────────────────────────────────────────────┘       │
│              │                                                          │
│              │    ┌─────────────────────────────────────────────┐       │
│              │    │  卡片3                                       │       │
│              │    │  说明文字（底部灰色小字）                    │       │
│              │    └─────────────────────────────────────────────┘       │
│              │                                                          │
│              │    ┌─────────────────────────────────────────────┐       │
│              │    │  卡片N                                       │       │
│              │    └─────────────────────────────────────────────┘       │
│              │                                                          │
│              │    ┌─────────────────────────────────────────────┐       │
│              │    │  [取消]                      [保存配置]      │       │
│              │    └─────────────────────────────────────────────┘       │
└──────────────┴──────────────────────────────────────────────────────────┘
```

## 13. 弹窗规范

### 13.1 确认弹窗

| 属性 | 值 |
|------|-----|
| 宽度 | 400px ~ 500px（最大不超过 90vw） |
| 圆角 | 12px |
| 阴影 | 0 25px 50px rgba(0,0,0,0.25) |
| 内边距 | 24px |
| 遮罩层 | rgba(0,0,0,0.5) |
| 按钮排列 | 取消在左，确认在右 |

### 13.2 成功/失败提示（Toast）

| 类型 | 样式规范 |
|------|----------|
| 成功提示 | 绿色图标 #10B981 + 成功文案，自动 3s 消失，可手动关闭 |
| 失败提示 | 红色图标 #EF4444 + 错误文案，需手动关闭 |
| 警告提示 | 黄色图标 #F59E0B + 警告文案，需手动关闭 |
| 信息提示 | 蓝色图标 #6366F1 + 信息文案，自动 5s 消失 |

### 13.3 删除操作规则

- 删除操作必须弹出确认弹窗
- 确认弹窗需包含：危险按钮样式（红色）+ 明确的操作说明
- 删除成功后显示成功 toast 提示
- 批量删除时显示数量，如"确定删除选中的 5 项？"
- 不可恢复的操作应在确认文案中说明"此操作不可恢复"
